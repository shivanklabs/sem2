
# Shiv Portfolio

A premium React portfolio project with a trading-inspired dark UI.

## Features

- React Router navigation
- Dark / Light mode
- Responsive design
- Animated hero section
- Project filtering
- Contact form
- Trading-inspired UI
- Modern cards and glassmorphism

## Tech Stack

- React JS
- React Router DOM
- Framer Motion
- React Icons
- CSS3

## Installation

npm install
npm run dev
